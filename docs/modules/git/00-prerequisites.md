# Prerequisites

| Tool | Why You Need It |

## Recommended Tools

| Tool | Why It Helps |
|------|--------------|
| [Visual Studio Code](../../tools/vscode.md) | Makes merge conflicts much easier to resolve with its built-in diff viewer |

None. This is a foundational module.
