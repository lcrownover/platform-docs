# Prerequisites

## Required Tools

| Tool | Why You Need It |
|------|-----------------|
| Terminal | You'll run Puppet and PDK commands here. macOS/Linux: use the built-in terminal. Windows: use [WSL](../../tools/wsl.md). |
| [Puppet Development Kit (PDK)](../../tools/pdk.md) | The official toolkit for creating and testing Puppet modules locally |
| Access to a Puppet server | You'll need a working Puppet infrastructure to test deployments. Installation is outside the scope of this module. |

## Recommended Tools

| Tool | Why It Helps |
|------|--------------|
| [Visual Studio Code](../../tools/vscode.md) | Syntax highlighting and linting for Puppet code with the Puppet extension |

## Required Modules

| Module | Why It's Required |
|--------|-------------------|
| [Git](../git/00-index.md) | Puppet code is infrastructure-as-code and should be version controlled. You'll need to understand commits, branches, and pull requests. |
