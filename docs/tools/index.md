# Tools

Reference guides for installing and configuring the tools used across our training modules.

Each guide covers installation, basic setup, and tips for getting the most out of the tool. **You don't need to install everything upfront**, check the prerequisites for each module to see what's required.

## Available Tools

| Tool | Platform | Description |
|------|----------|-------------|
| [Homebrew](homebrew.md) | macOS | Package manager for installing CLI tools |
| [Windows Terminal](windows-terminal.md) | Windows | Modern terminal with tabs, profiles, and WSL integration |
| [WSL](wsl.md) | Windows | Run Linux on Windows—recommended for development |
| [Visual Studio Code](vscode.md) | All | Code editor with Git integration and remote development |
| [Git](git.md) | All | Version control system |
| [Command-Line Editors](cli-editors.md) | Linux / macOS | Terminal-based text editors (nano and vim) for SSH sessions |
| [Ruby](ruby.md) | All | Required for Puppet ecosystem tools like puppet-lint and eyaml |

## Puppet Tools

| Tool | Platform | Description |
|------|----------|-------------|
| [Puppet VS Code Extension](puppet-vscode.md) | All | Syntax highlighting, autocompletion, and linting for Puppet code in VS Code |
| [puppet-lint](puppet-lint.md) | All | Style guide checker for Puppet manifests |
| [eyaml](eyaml.md) | All | Encrypt secrets in Hiera data files for safe version control storage |
| [Puppet Development Kit (PDK)](pdk.md) | All | Build and test Puppet modules |
